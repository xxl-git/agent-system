### 发现的类型不一致
`SubTask` 接口在 `src/core/task-decomposer.ts` 和 `packages/resilience/src/types.ts` 中不同：
- SRC: `dependsOn: string[]; result?: { success, output, error }; estimatedMinutes: number`
- PKG: `dependencies: string[]; result?: string; error?: string; retries?: number; priority?: number`
- 这导致 `orchestrator.ts` 的 `getCheckpointManager()` 调用产生类型错误
- 工作区：回退 orchestrator.ts 的 resilience import 到相对路径
