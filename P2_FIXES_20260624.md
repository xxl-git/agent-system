## ② safePath() 防御增强
**修复文件**: `src/server/agent-server.ts`
**原问题**: `decodeURIComponent(input)` 遇到无效编码(如 `%ZZ`)抛 `URIError`，导致服务器 500 崩溃
**修复方案**: 
- `decodeURIComponent()` 包裹 try/catch，解码失败时返回 null 并记录 WARN 日志
- 调用方收到 null 后返回 403 Forbidden
**测试结果**:
| 请求 | 预期 | 实际 | 日志 |
|------|------|------|------|
| `/%ZZ` | 403 (非崩溃) | ✅ HTTP 403 Forbidden | `[safePath] URL 解码失败，拒绝访问: /%ZZ` |
| `/` | 200 | ✅ HTTP 200 OK | 正常 |
| `/%2e%2e%2f%2e%2e%2fetc` | 403 (目录穿越) | ✅ HTTP 403 Forbidden | 正常拦截 |
