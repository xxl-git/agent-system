# P7: 专业聊天界面 (agent-ui.html v2)

**日期**: 2026-06-18
**状态**: ✅ 完成

## 变更摘要

从简约聊天界面全面升级为**专业级聊天 UI**，参考 OpenClaw Control UI 设计风格。

### 之前
- 基础消息气泡 + 输入框
- 无模型信息展示
- 无消息操作（不能复制/重试/编辑/删除）
- 无设置面板
- 无上下文指标
- 无主题切换
- 消息不持久化（刷新丢失）

### 之后
| 功能 | 说明 |
|------|------|
| **模型标签** | Header 显示当前模型 + 本地/远程标签（颜色区分） |
| **Status Bar** | 连接状态 · 模型名 · 会话ID · 运行时间 · 上下文进度条 |
| **模型选择器** | 下拉菜单列出所有可用模型，本地/远程标识，点击切换 |
| **消息操作** | Hover 显示复制/重新生成/编辑/删除按钮 |
| **设置面板** | 侧滑抽屉，配置模型/API地址/超时/重试/Token/主题 |
| **停止生成** | AbortController 中断请求 |
| **消息持久化** | localStorage 存储最后100条，刷新不丢 |
| **主题切换** | 暗色/亮色/跟随系统，CSS 变量驱动 |
| **字符计数** | 输入框右下角实时显示 |
| **Agent 模型标签** | 每条 Agent 回复底部显示所用模型 |
| **快捷命令** | /status /help /models /project list /summarize /memory search |

### 设计要点
- 色彩：深色背景(#0f0f13) + 浅蓝色强调(#6366f1) + 绿/红/黄状态色
- 气泡：用户(紫色渐变右对齐) / Agent(深色边框左对齐) / 系统(居中透明小字)
- 动画：消息淡入 + 打字闪光
- 滚动条：细条透明（聊天区 hover 显示）
- 响应式：移动端自动折叠导航、缩小气泡

## 后端 API 依赖
- `GET /api/models` — 获取可用模型列表 → 选择器 + 设置下拉
- `GET /api/config` — 获取当前配置 → 设置面板预填
- `POST /api/config` — 保存配置变更 → 写入 default.json
- `GET /api/status` — 获取连接状态/模型/会话ID
- `GET /api/events` — SSE 连接状态感知
