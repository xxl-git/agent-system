## 完成内容

### P1-1: 跨会话记忆恢复 (`src/memory/session-recovery.ts`)

- `SessionRecoverer` 类：从 sql.js DB 加载最近决策 + 跟踪实体 + 会话摘要
- 从 `memory/` 目录加载最近N天文件记忆

### P1-2: 审计日志 (`src/audit/audit-log.ts`)
**功能**：记录所有决策/工具调用/错误/会话边界，按日持久化
- `AuditLog` 类：8种事件分类 + 5种结果状态
- 便捷方法：logToolCall / logDecision / logModelSwitch / logRecovery / logDegradation / logCheckpoint / logError
- 查询能力：按分类/结果/会话/时间/关键词过滤
- 轮转机制：单文件5MB上限，保留5个轮转文件
- 16项测试全部通过（含边界测试 + 集成验证）

### Agent Core 集成
**修改**: `src/core/agent/agent-core.ts`
1. **init()**: 启动时调用 `SessionRecoverer.recover()` → 注入历史记忆到 system prompt

5. **新命令**: `/audit summary|recent|errors|search` + `/memory status|decisions|entities|reload`
