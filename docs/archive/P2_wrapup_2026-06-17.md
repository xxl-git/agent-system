## 完成内容

### P2: 记忆摘要引擎 (`src/memory/summarizer.ts`)
**功能**：自动将对话蒸馏为结构化知识，支持规则引擎 + LLM 双模式
- `MemorySummarizer` 类（~380行）：
  - **summarizeSession()**: 会话结束时自动生成结构化摘要
  - **patrolSummary()**: 长会话中途快照（心跳触发，>20条消息阈值）
  - **backfill()**: 历史会话批量补摘要
  - **getSummaries()/getRecentSummaries()**: 多维度查询
- 摘要输出 7 个维度：sessionSummary、keyDecisions、learnedFacts、entityUpdates、tags、nextSteps、knowledgePoints
- 规则引擎：中英混合分词（2-gram + 单字CJK）、事实提取（正则匹配 ✅/已完成/创建了/修复了）、实体检测（模型名/文件路径/模块名）、标签映射（20+中文→英文映射表）、下一步推断
- LLM 模式：用 lmstudio adapter 生成 JSON 摘要，失败自动回退规则引擎
- 持久化：写入 summaries 表，上限50条自动清理旧记录

### 集成
- **agent-core.ts**:
  - init(): 初始化 summarizer 实例
  - stop(): 会话结束自动触发摘要
  - handleCommand(): /summarize now|list|recent|patrol 命令
- **db-store.ts**: 新增 querySummaries()、queryRecentSummaries()、queryDecisions()、pruneSummaries()、storeSummary()

### 分词改进
- 修复：原中文单字全被 w.length<2 过滤 → 改为 CJK 单字可通过 + 词组自动展开 2-gram
- 修正前：中文消息无法提取任何主题词 → 修正后：有效提取「帮我创建」「记忆摘要」「引擎」等关键词
